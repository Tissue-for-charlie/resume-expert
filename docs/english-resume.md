# 英文简历专项指南

> 本文是 SKILL.md 中英文简历规则的扩展阅读，包含更详细的模板和常见错误。

## STAR 框架实战

STAR = Situation（情境）→ Task（任务）→ Action（行动）→ Result（结果）

### 例：后端开发经历

**Task 清单风格（❌）**：
> Responsible for developing the payment module. Used Spring Boot and MySQL. Fixed several bugs.

**STAR 框架重写（✅）**：
> Migrated the payment module from a monolithic architecture to microservices (Spring Boot + MySQL), reducing transaction latency by 40% and eliminating 3 recurring production incidents. Designed a distributed transaction mechanism with idempotency guarantees for 50K+ daily payment requests.

对比要点：
- "Responsible for" → "Migrated"（主动动词）
- "Used X" → 技术在叙事中出现（"microservices (Spring Boot + MySQL)"）
- "Fixed bugs" → "eliminated 3 recurring incidents"（量化）
- 无结果 → "reducing latency by 40%"（量化结果）

## 动词时态速查

| 描述对象 | 时态 | 示例 |
|----------|------|------|
| 当前/最近职位 | 现在时 | Designs, Implements, Leads |
| 过往职位/已完成项目 | 过去时 | Designed, Implemented, Led |
| 持续维护的项目 | 现在时 | Maintains, Optimizes |
| 一次性成果 | 过去时 | Launched, Migrated, Reduced |

## 常见错误

### 1. 用代词 "I" / "My"

❌ "I designed the API and I implemented the caching layer"

✅ "Designed the API and implemented a Redis caching layer, reducing latency by 60%"

### 2. 被动语态

❌ "The system was designed to handle 10K concurrent users"

✅ "Designed the system to handle 10K concurrent users"

### 3. 弱动词

| ❌ 弱动词 | ✅ 替换 |
|-----------|--------|
| Worked on | Developed, Built, Implemented |
| Helped with | Contributed to, Co-developed |
| Was part of | Collaborated on, Joined |
| Did | Performed, Executed, Completed |
| Made | Created, Developed, Produced |
| Got | Achieved, Attained, Secured |

### 4. 动词重复

避免 bullet 都用同一个动词开头：

❌
- Implemented user authentication
- Implemented payment module
- Implemented CI/CD pipeline

✅
- Engineered a JWT-based authentication system
- Built a payment module with Stripe integration
- Automated CI/CD pipeline with GitHub Actions

### 5. 中文习惯直译

❌ "负责系统开发工作" → "Responsible for system development work"

✅ "Developed the core system, including..."

"负责"在英文中没有直接对应——英文简历用动词开头而非描述职责。
